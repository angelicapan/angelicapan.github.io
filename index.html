Work and Involvement

This Summer, I was very lucky to have joined the engineering team at Hitachi Vantara's headquarters in Santa Clara, where I developed fully functional user-facing features such as baremetal provisioning and server replacement modules using React.js, Redux Sagas, and the Axios library to handle requests between REST endpoints within server, network and storage systems within UCP systems. Features will be launched in Hitachi’s Unified Compute Platform Advisor HTML5 product release by end of 2019 (I am very excited!)

As lead for this year's CSK, I help define CS Kickstart’s direction through leading board meetings, maintaining department relations with UC Berkeley’s College of Engineering, establishing and maintaining relations with sponsor companies in The Bay Area, as well as directing community outreach to expand CS Kickstart to other universities. I also work with the curriculum team at CS Kickstart, where I help create a project-based technical curriculum in Web Development, Python, and Graphic Design for CS Kickstart’s ~60 participants.

In Summer of 2018, I had the pleasure of interning as an associate product manager at PT Bizzy E-Commerce in Central Jakarta in Indonesia, a B2B E-Commerce company, where I consulted external departments and company clients to define future sprint plans as well as create wireframes and mockups for in-platform messaging and editable purchase orders on Bizzy’s vendor facing platform, maximizing the efficiency of approval flow in the procurement process for Bizzy’s vendors.

As Head of Design, I define our student association’s branding and online presence through designing and creating visual assets and promotional media using Adobe Photoshop/Illustrator, as well as design for BISA’s ’Nusantara’, now the largest annual Indonesian cultural festival in The Bay Area.

Technical Projects

Portfolio

Pay Pay Web App

PayPay aims to simplify the logistical organization of sales events and increase customer and vendor satisfaction by providing an intuitive ordering and payment verification web platform to reduce queues and remove unnecessary use of physical tickets.

Monsters! The Game

Used Java and utilized Java’s StdDraw library to create a game that generates random explorable worlds based on a given seed and supports several player interactions based on command line argument and GUI user input. Players must unlock all the doors in time before the monsters catch them!

Chrome for Social Good

A chrome extension to raise awareness to blind consumerism by using natural language processing to assess and summarize a company's controversial practices or contributions to social and environmental initiatives as you shop. By associating each company/product to a 'Social Good Rating', we also hope to reward companies with worthy causes and incentivize others to work towards ethical practices. Built with Python, Javascript, and the help of MediaWiki and Google Cloud's Natural Language API.

Read more about this project!

Google Maps Simulator

Built routing and rendering engine for web mapping application that supports point-to-point navigation for Berkeley area based on the A* algorithm. Used Java along with data extracted from OpenStreetMap project
